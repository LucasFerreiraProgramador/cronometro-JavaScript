# Cronômetro em JavaScript

Um cronômetro interativo desenvolvido com foco em manipulação de tempo, controle de estado e interações com o usuário utilizando JavaScript puro.  
O projeto permite iniciar, pausar e resetar a contagem de tempo em tempo real.

🔗 Acesse o projeto:  
https://lucasferreiraprogramador.github.io/cronometro-JavaScript/

---

## Funcionalidades

- Iniciar contagem de tempo  
- Pausar cronômetro  
- Resetar contagem  
- Atualização em tempo real  
- Interface simples e objetiva  

---

## Tecnologias Utilizadas

- HTML5 — estrutura semântica  
- CSS3 — layout e estilização  
- JavaScript (Vanilla) — lógica do cronômetro
