## 🧩 Sobre o projeto

Aplicação simples de cronômetro desenvolvida com JavaScript,
com foco em lógica de programação, manipulação do DOM e controle de tempo.

O projeto foi criado para praticar conceitos fundamentais de front-end,
como eventos, temporização e atualização dinâmica da interface.

---

## 🎯 Funcionalidades

- Iniciar a contagem do tempo
- Pausar o cronômetro
- Zerar a contagem
- Exibição do tempo em tempo real (HH:MM:SS)

---

## 🛠 Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript
